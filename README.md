# Portfolio

## Overview
This repository contains the source code of my portfolio website.

**Live Link:** [http://sarwar76200.netlify.app](http://sarwar76200.netlify.app)


## Tech Stack
- React 18
- Next.js
- shadcn/ui
- Lucide Icons


## Features
- Beautiful and modern ux with shadcn/ui
- Fully responsive design, adapts to any screen size
- Light and dark mode animation with ViewTransition API
- Clean project structure, easily maintainable


## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sarwar76200/Portfolio.git
   cd Portfolio
   ```
2. Install dependencies:

   ```bash
   npm install #or pnpm install
   ```
3. Run the applications:

   ```bash
   npm run dev #or pnpm dev
   ```
4. Navigate to the URL shown in the terminal (http://localhost:3000)
